# 비개발자용 바이브 코딩 올인원 교육 플랫폼

**AI 협업 코딩 학습 및 콘텐츠 실시간 관리 포탈**

비개발자도 AI 어시스턴트와 함께 소프트웨어를 직접 기획하고 코딩할 수 있는 **'바이브 코딩'** 교육 플랫폼입니다.

---

## ✨ 주요 기능

### 📖 커리큘럼 마스터
- 단원별 과정 소개 및 실습 가이드 (마크다운 지원)
- PPTX / PDF 슬라이드 파일 업로드 및 페이지 탐색 뷰어
- 강사 슬라이드 노트 자동 추출 → 자율학습 가이드 연동
- 관리자 단원별 슬라이드 노출 여부 제어 (교육 진행에 맞춰 순차 공개)

### 🔐 회원 / 인증
- 회원가입 및 로그인 (쿠키 기반 7일 세션 유지, 새로고침해도 로그인 유지)
- 관리자 계정 자동 부여 (ID: `admin` 또는 성명: `관리자`)
- 비밀번호 변경 기능

### 💬 실시간 Q&A
- 로그인한 누구든 질문 및 다중 답변 가능
- 하나의 질문에 여러 답변 누적 저장
- 관리자는 질문 개별 삭제 가능

### 👥 관리자 전용 패널
- 전체 회원 목록 조회 및 개별 삭제
- 전체 Q&A 레코드 조회 및 삭제 (개별 / 전체 초기화)

## 🚀 로컬 실행 방법

### 1. 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. 앱 실행
```bash
streamlit run main.py
```

### 3. 접속
브라우저에서 `http://localhost:8501` 접속

## 📁 프로젝트 구조

```
vibedu/
├── main.py              # Streamlit 앱 메인
├── database.py          # 파일 기반 DB 헬퍼 (JSON)
├── requirements.txt     # 의존 패키지
├── curriculum.json      # 단원/슬라이드 데이터 (자동 생성)
├── users.json           # 회원 정보 (자동 생성)
├── qna.json             # 질의응답 레코드 (자동 생성)
└── slides/              # 업로드된 슬라이드 파일 (자동 생성)
```

> **참고**: `users.json`, `qna.json`, `curriculum.json`, `slides/` 는 `.gitignore`에 포함되어 있으며 Git에 업로드되지 않습니다.

## ⚙️ 관리자 계정 설정

앱 최초 실행 후 회원가입 시 다음 조건 중 하나를 충족하면 자동으로 관리자 권한이 부여됩니다:
- **아이디**를 `admin`으로 입력
- **성명**을 `관리자`로 입력

## 🛠️ 기술 스택

| 항목 | 기술 |
|------|------|
| 프레임워크 | Streamlit |
| PDF 슬라이드 렌더링 | PyMuPDF (fitz) |
| PPTX 처리 | python-pptx |
| 세션 유지 | extra-streamlit-components (CookieManager) |
| PPTX → PDF 변환 | win32com (Windows 환경, PowerPoint 필요) |
| 데이터 저장 | JSON 파일 시스템 (No DB) |
